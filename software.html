This page contains software developed in our group at Simon Fraser University, as well as links to software and websites developed in collaboration with other groups.
Github. The most recent versions of the software developed in our lab and with our collaborators are available from our GitHub repository. The main tools available there and on some of our collaborators github sites:
  • Pathogens genomics
    • MentaLiST, frugal and accurate Multi-Locus Sequence Typing: MentaLiST.
    • PRINCE, calling Copy-Number Variants from NGS data: PRINCE.
    • HyAsP, a hybrid assembler for plasmids: HyAsP.
  • NGS tools
    • CoLoRMap, hybrid long reads correction: CoLoRMap.
    • LRCstats, evaluating long reads correction tools: LRCstats.
    • Calib, clustering UMI-barcoded sequencing data: Calib.
  • Comparative genomics
    • ANGeS, reconstructing ancestral genome maps: ANGeS.
    • SCJ-TD-FD, the Single-Cut-or-Join distance with duplicated genes: SCJ-TD-FD.
    • PhySca, the Single-Cut-and-Join (SCJ) small parsimony problem for weighted gene adjacencies: PhySca.
    • DeCoStar, reconstructing parsimonious histories of gene adjacencies: DeCoStar (see also DeCoSTAR_pipeline for an application to comparative scaffolding).
    • DeClone, Evolutionary scenarios for gene adjacencies in a Gibbs-Boltzmann framework : DeClone.
    • ecceTERA, gene tree species tree reconciliation: ecceTERA
  • Paleogenomics and ancient DNA
    • FPSAC, fast phylogenetic scaffolding of ancient contigs: FPSAC.
    • AGaPeS, comparative gap filling in ancient DNA assemblies: AGapEs.
    • EWRA, Edge-Weighted Reconstruction integrating aDNA Assembly graph: EWRA.
  • Other
    • flowLearn, a machine-learning algorithm for gating flow cytometry data: flowLearn.
    • RNA-unchained, Chaining sequence/structure seeds for computing RNA similarity: RNA-unchained.
